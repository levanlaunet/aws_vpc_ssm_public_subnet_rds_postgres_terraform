# aws_vpc_ssm_public_subnet_rds_postgres_terraform
## AWS VPC + SSM Public Subnet + RDS Postgres + Terraform example
### ssh-keygen -t rsa -b 4096 -C "myemail@gmail.com"
### terraform init
### terraform plan [--var-file "terraform_dev.tfvars"]
